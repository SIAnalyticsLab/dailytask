🏈 SI Analytics 30-Day Data Visualization Challenge — Daily Task 1: NFL Injuries Dashboard

📘 Background

In this task, you’ll explore the NFL Injuries dataset compiled by JJ Allaire. The dataset tracks players' injury status — Questionable, Out, or Injured — across multiple years in the NFL.

📂 Dataset: nfl-injuries.csv
Each row represents a player’s weekly injury record and contains:
• data: Injury status (e.g., Out, Questionable)
• year: Season year (2012–2022)
• row_number: Entry index (used for counting or tracking records)

🎯 Objective

Recreate a simple yet insightful dashboard similar to the sample below, showing yearly trends and summary insights for player injuries.

🧮 Step-by-Step Instructions

  1. Load and Inspect the Data
    • R: readr::read_csv("nfl-injuries.csv")
    • Python: pd.read_csv("nfl-injuries.csv")
    • Power BI: Home → Get Data → Web → paste the GitHub CSV URL above.
    Preview columns and confirm that the dataset has data, year, and row_number.
  2. Prepare and Summarize
    Group the data by year and injury status, then compute:
    • Count of players per status per year
    • Average count across years (optional metric)
    Example summary: 
    Year   Status         Count
2012   Out            480
2012   Questionable   510
  3. Build Your Dashboard
    Create the following visuals:
    • 🧮 Value Boxes / Cards:
      • Average Injured
      • Average Out
      • Average Questionable
    • 📈 Line Chart: Number of players by status over years (compare trends).
    • 📊 Bar Chart: Total count of injuries by year.
    💡 R/Python users: Use Quarto Dashboard or Plotly.
    💡 Power BI users: Combine Cards, Line, and Bar charts into a single-page report.
  4. Add Design Features
    • Apply distinct colors for each injury status.
    • Add a title: “NFL Injuries — 10 Years Trend Dashboard”.
    • Keep layout clean and professional with readable labels.
