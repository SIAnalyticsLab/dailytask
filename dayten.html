💳 SI Analytics 30-Day Data Visualization Challenge — Day 10

Bank Transactions Dataset: Fraud Detection & Account Flow Dashboard
📁 Dataset: bank.xlsx

🧩 Background

Today’s dataset — “bank.xlsx” — contains extracted transaction records from multiple bank accounts. It captures real financial movements, including deposits, withdrawals, cheque references, and running balances.

The data provides an excellent opportunity to explore financial transaction visualization, fraud detection analytics, and cash flow tracking — essential skills for analysts working in finance, compliance, and data auditing.

📄 Dataset name: bank.xlsx
📍 Source: SI Analytics GitHub Repository

🎯 Daily Task Objective

Build a Bank Transaction Dashboard that answers:

💡 Core Question: “How do deposits, withdrawals, and balances evolve over time — and which transactions or accounts show unusual patterns?”

You’ll visualize transaction flows, balances, and inflows/outflows across accounts to identify anomalies, large transactions, and potential red flags.

🧮 Step-by-Step Instructions

  1. Load and Explore the Data
    • R: readxl::read_excel("bank.xlsx")
    • Python: pd.read_excel("bank.xlsx")
    • Power BI: Home → Get Data → Excel → Select the file
    Inspect for missing values and confirm numeric types for withdrawal, deposit, and balance columns. Convert Account No to categorical if needed.
  2. Clean and Prepare
    • Convert Date and Value Date to date formats.
    • Create new fields:
    • 💰 Net Transaction = Deposit – Withdrawal
    • 📈 Cumulative Balance = Running total per account
    • ⚠️ Large Transaction Flag = Transactions > ₹500,000
    • Group by account to calculate total deposits, withdrawals, and average transaction value.
  3. Build Your Dashboard
    Use R, Python, or Power BI to visualize:
    • 💳 Line Chart: Account balance trend over time
    • 💸 Bar Chart: Total deposits vs withdrawals per account
    • 🔍 Table or Card: Top 10 largest transactions
    • 🧭 Stacked Column: Monthly inflow vs outflow
    • ⚠️ Scatter Plot: Deposit vs withdrawal (flag outliers)
    💡 R/Python Tip: Use color gradients (green → red) to flag high-value withdrawals.
    💡 Power BI Tip: Add slicers for “Account No” and “Transaction Type” for interactivity.
  4. Add Key Metrics / KPI Cards
    Display summary indicators such as:
    • 💰 Total Deposits
    • 💸 Total Withdrawals
    • 🏦 Average Balance
    • ⚠️ Number of High-Value Transactions (> ₹500,000)
    • 🧾 Accounts with Negative Balances
