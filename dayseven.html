💳 SI Analytics 30-Day Data Visualization Challenge — Day 7

Loan Portfolio Risk and Repayment Patterns
📁 Dataset: loans_full_schema.csv

🧩 Background

Today’s dataset — “loans_full_schema.csv” — is adapted from Lending Club’s public lending data. It contains detailed records on personal and business loans, including borrower demographics, credit histories, income verification, and repayment performance.

The goal is to analyze how borrower characteristics relate to loan grade, interest rate, and repayment success — key skills in financial risk analytics.

📄 Dataset name: loans_full_schema.csv

🎯 Daily Task Objective

Build a Loan Portfolio Dashboard that answers:

💡 Core Question: “What factors influence loan repayment performance and interest rates?”

You’ll analyze how borrower risk, income verification, and loan grade impact repayment — and identify which loan purposes carry higher default risk.

🧮 Step-by-Step Instructions

  1. Load and Explore the Dataset
    • R: read_csv("loans_full_schema.csv")
    • Python: pd.read_csv("loans_full_schema.csv")
    • Power BI: Home → Get Data → Text/CSV → Select the file
    Inspect missing values and confirm data types for numeric columns such as annual_income, interest_rate, and loan_amount.
  2. Clean and Prepare the Data
    • Create derived metrics:
    • 🏦 Loan-to-Income Ratio = loan_amount ÷ annual_income
    • 💰 Interest Burden Index = interest_rate × debt_to_income
    • 📈 Repayment Ratio = paid_principal ÷ loan_amount
    • Filter for relevant statuses (e.g., Current, Fully Paid, Charged Off).
    • Group or summarize by loan grade, state, or homeownership type.
  3. Build Your Dashboard
    Use R, Python, or Power BI to visualize:
    • 📊 Bar Chart: Average interest rate by loan grade (A–G)
    • 💵 Pie Chart: Distribution of loan purposes (debt consolidation, home improvement, etc.)
    • 📈 Scatter Plot: Loan amount vs. interest rate (colored by grade)
    • 🧭 Stacked Column Chart: Loan status by application type (individual vs. joint)
    • 📆 Line Chart: Monthly loan issuance trends by grade (issue_month)
    💡 Optional (Advanced): Add slicers or filters for loan_status or state to make your dashboard interactive.
  4. Add Key Performance Cards
    Display 3–4 summary metrics such as:
    • 🏠 Average Loan Amount
    • 💳 Average Interest Rate
    • 💰 Total Loans Fully Paid
    • ⚠️ % of Loans in Default or Late
